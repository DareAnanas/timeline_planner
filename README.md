# Timeline Planner

## Про проєкт
Це веб-застосунок для планування, який використовує парадигму відеомонтажу. Кожне завдання це кліп, який ти можеш розмістити на таймлайні. Час, який не заповнений завданнями залишається вільним часом. Таким чином, ти можеш бачити скільки у тебе вільного часу та загальну картину.

## Ключові функції (MVP)
- **Магнітний таймлайн:** Завдання притягуються до днів.
- **Динамічне стиснення:** Якщо на один день припадає кілька завдань, вони ділять простір і змінюють колір (Зелений -> Червоний), сигналізуючи про навантаження.
- **Ripple Edit:** Вставка завдання автоматично зсуває всі наступні дедлайни (як insert mode у Davinci Resolve).
- **Абстрактні шари:** Можливість додавати абстрактні завдання поверх основного потоку.

## Технологічний стек
- **Core:** React, TypeScript, Vite
- **State:** Zustand
- **Drag & Drop:** @dnd-kit/core
- **Styling:** Tailwind CSS

## Як запустити проєкт

1. **Клонувати репозиторій**
```bash
git clone https://github.com/DareAnanas/timeline_planner
```
2. **Встановити залежності**
```bash
npm install
```
3. **Запустити локальний сервер**
```bash
npm run dev
```
Веб-додаток доступний за адресою `http://localhost:5173`
